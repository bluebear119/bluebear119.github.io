# OTCGN 介紹

一段介紹

# 安裝過程

# 下載 OCTGN

可以到官方網站進行下載，下載前記得確認電腦滿足相關的前置條件。

# 安裝 Arkham Horror:The Card Game Definition

一旦你安裝好 OTCGN 之後，你就可以打開這個軟件，選擇 Games Manager，找到 Arkham Horror:The Card Game Definition 後，點擊安裝，之後 OTCGN 就會直接自動替你安裝好這個桌遊。

但是，安裝好的桌遊並不包含遊戲圖片文件，這時候你需要到這個頁面去下載相關的圖片包。

# 安裝圖片包

在 Game Manager 裡面，選擇 Add Image Pack，之後把剛剛下載好的圖片包加進去

# 進行遊戲前期準備

想開始玩遊戲，你需要點擊 Play and Spectate，再按下 Start 按鈕。

看到這個頁面後再按一次 Start

成功進入遊戲主畫面後你會見到這樣的畫面。

你需要做的第一件事是：載入調查員的牌組。
按下 Ctrl+L 或者在選擇 Game > Load Deck, 之後選擇你想使用的調查員相關的牌組的.o8d 文件，比方說我想用 Stella Clark 作為今次的調查員，選擇後，OCTGN 就會替你載入這幅牌組。

你也可以在 Arkham DB 裡面找到其他人所建立的牌組，如果找到你喜歡想使用的也可以直接下載之後再載入 OTCGN。

接下來，你需要載入的就是和遊戲劇情相關的牌組，這包括了遭遇牌組，場景牌組以及密謀牌組。選擇 Game > Load Prebuild Deck , 選擇你今次要跑劇本對應的牌組。

例子：想跑的劇本為 The Path To Carcosa 的第一個劇本：Curtain Call.

那我要 Load 的 Deck 就可以在下面的路徑找到。
C:\Users\yourpcname\AppData\Local\Programs\OCTGN\Data\GameDatabase\a6d114c7-2e2a-4896-ad8c-0330605c90bf\Decks\03 - The Path to Carcosa

之後可以根據劇本指引來對這些卡片進行位置的設定。可以到官方網站找到相關劇本裡面指引去設定特定的遊戲場景

也就是說我該下載 The Path To Carcosa 的劇本指南。

按照劇本指示擺放好遊戲的地點卡牌的位置例如這樣。

劇本提到的遭遇牌會預先洗好好放在下面這個位置上

而劇本提到的 Set Aside 的牌，你需要按下 Global 才可以見到。

拉出來驗證一下看是不是劇本提到的都有在。檢查完畢發現沒有遺漏。

# 正式遊戲

遊戲開始的時候的第一個階段會是調查階段，點擊你角色欄位框框的播放符號就會進入調查階段。

把你的角色拖到遊戲起始地點（劇本會告訴你起始地點），然之後你就可以翻開你身處地點的地點卡。

右點擊你的卡牌或者 Ctrl+F 你就可以翻轉地點卡的另一面，閱讀上面文字內容，若有需要就可以放置相關數量的線索。

在地點卡上右點擊可以加入或者移除 Token，包括資源 Token，線索 Token, 傷害 Token，恐怖 Token，行動 Token 以及毀滅 Token。
