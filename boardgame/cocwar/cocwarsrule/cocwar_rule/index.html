# 把地圖放好

  • 遊戲裡面有兩塊地圖分別為東大陸和西大陸，每一塊大陸都會印有兩面。
  • 這兩面的地形是一樣的，但 5 人地圖的地域分佈會比 3 人地圖的地域分佈來的更為仔細。
