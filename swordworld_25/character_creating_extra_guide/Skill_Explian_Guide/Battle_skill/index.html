# 戰鬥特技 - 常時特技

《閃避》／《かいくぐり》
  • 前提：無
  • 使用：《盾》
  • 概要：近戰時裝備盾牌並迴避成功時，下次攻擊 C 數值 -1
  • 效果：以盾牌架開攻擊後，使自己取得有利位置並進行致命的一擊。習得這個特技的角色，要記下在一回合內對來敵人的近接攻擊的回避力判定成功的次數，自己下一個回合進行的第一次近接攻擊時，暴擊值會「- 回避成功的次數」，但不會使暴擊值降到 7 或以下。
  • 角色自己進行的攻擊，如果是對多個目標進行的攻擊的話，則從命中的目標中選 1 個，在決定傷害時適用此效果。
  • 此效果在迴避力判定時必須裝備盾，而在進行近接攻擊時沒有盾也無妨。此外，只要進行 1 次接近攻擊就會失去效果。
《回避行動 I》／《回避行動 I》
  • 前提：冒險者等級 3 以上
  • 使用：無
  • 概要: 回避力判定 +1
  • 效果：回避力常駐 +1。
《終律增強》／《終律増強》
  • 前提：吟遊詩人技能等級 3 以上
  • 使用：吟遊詩人技能
  • 概要：終律的威力 +10
  • 效果：終律造成的傷害和回復量的威力 +10
《咒歌追加 I》／《呪歌追加 I》
  • 前提：吟遊詩人技能等級 1 以上
  • 使用: -
  • 概要：咒歌和終律的習得數 +1
  • 效果：可學習的咒歌和終律的數量 +1。換言之，共計可學習的咒歌和終律的數量為「吟遊詩人技能等級 +1」。
